# Bingo Display

A local web application for running bingo games with a projected display page and an admin control page.

## Install

```bash
npm install
```

## Run

```bash
npm start
```

Then open:

- Admin page: http://localhost:3000/admin
- Display page: http://localhost:3000/display

For development with automatic server restarts:

```bash
npm run dev
```

## Notes

Game state is stored in memory. Restarting the server will clear the current game.
